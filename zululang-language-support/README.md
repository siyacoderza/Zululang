# Zululang Language Support

VS Code language extension for Zululang programming language.

## Features

- Syntax highlighting for `.zul` and `.zulu` files
- Support for Zululang keywords (uma, kungenjalo, ngenkathi, umsebenzi, buyisa, kuba, bonisa, cela)
- String highlighting
- Numeric constant highlighting

## Files

- `syntaxes/zululang.tmLanguage.json` - TextMate grammar definition
- `language-configuration.json` - Language configuration (comments, brackets, etc.)
- `package.json` - VS Code extension manifest
